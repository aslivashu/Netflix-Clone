A pixel-perfect, responsive front-end clone of the Netflix landing page. This project replicates the iconic user interface of the streaming giant, featuring interactive elements like the FAQ accordion and a fully responsive layout that adapts to mobile, tablet, and desktop screens.

## ‚ú® Features
-   **Responsive Design:** specific media queries (`max-width: 950px`, `768px`, `550px`, `375px`) ensure the layout looks great on any device, from large desktops to mobile phones.
-   **Interactive FAQ Section:** A functional accordion menu built with JavaScript that expands and collapses frequently asked questions.
-   **Visual Effects:** Includes the signature dark theme, hero background image with vignette overlay, and "Download" animation styles.
-   **Language Dropdown:** A toggleable language selection menu in the footer.
-   **Dynamic Headers:** Sticky and transparent header styling elements (simulated).
